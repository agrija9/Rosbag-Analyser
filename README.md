## Prerequisites

Software needed to run this repository:

- Python 3.6
- [Ubuntu 16.04](https://ubuntu.com/download/desktop)

## Installation

### Clone

Clone this repository in you local machine

```

```

### Setup

```
pip install -r requirements.txt
```
